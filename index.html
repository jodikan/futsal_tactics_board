📖 이용 방법
① 배치
+ 팀 색상 선택 후 선수 추가 클릭 → 코트 원하는 위치 클릭
+ 같은 색 = 같은 팀, 번호는 팀별로 자동 부여
+ 공 추가로 공 위치도 지정 가능
+ 선택/이동으로 위치 드래그 조정
② 전술 흐름 만들기
+ 프레임 추가 → 선수 위치 변경 → 반복
+ 편집 중엔 다음 프레임 이동 방향이 점선 화살표로 미리 표시됨
+ 타임라인 점 클릭으로 프레임 간 이동
+ 프레임은 최소 2개 이상 있어야 움직임이 보임
③ 재생 & 공유
+ 재생 버튼으로 애니메이션 미리 확인
+ 영상 저장으로 WebM/MP4 파일 다운로드 (Chrome·Edge 권장)
+ 저장된 영상은 카톡·단톡방에 파일로 공유 가능
+ 녹화 중 화면이 잠깐 재생됩니다 — 조작하지 마세요
